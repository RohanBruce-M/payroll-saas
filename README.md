# Payroll SaaS Application
## Change into dev branch to see the actual folder structure.
## Description
A web-based Payroll SaaS system that allows companies to manage employees, calculate salaries, and generate payslips securely.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js (Express)  
  *(Planned migration to Java Spring Boot)*
- Database: MySQL

## Project Structure
The project follows a clear separation of concerns:

- frontend/  → Frontend application built using React and Vite
- backend/   → Backend REST API built using Node.js and Express
- database/  → Database schemas, migrations, and related files
