# Payroll SaaS Application

## Description
A web-based Payroll SaaS system that allows companies to manage employees, calculate salaries, and generate payslips securely.

## Tech Stack
- Frontend: React
- Backend: Java (Spring Boot)
- Database: MySQL

## Project Structure
The project follows a clear separation of concerns:

- frontend/  → Frontend (React application)
- backend/   → Backend (Java Spring Boot application)
- database/  → Database-related scripts and configurations
